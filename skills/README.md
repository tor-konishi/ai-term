# スキルファイル

このディレクトリには、ネットワーク機器ベンダー毎のスキルファイルを格納します。

## ディレクトリ構造

```
skills/
├── cisco/
│   └── ios.md
├── vyos/
│   └── vyos.md
└── README.md
```

## スキルファイルの作成方法

1. ベンダー名のフォルダを作成（例: `cisco`, `vyos`, `juniper`）
2. フォルダ内にマークダウンファイル（.md）を作成
3. 以下の形式で設定内容を記述：

```markdown
# ベンダー名 スキル

## カテゴリ名

### 設定項目名
\```
コマンド例
\```

### 説明
設定の解説
```

## 自動検出キーワード

AIへの質問に以下のキーワードが含まれると、自動的に該当スキルが読み込まれます：

- **Cisco**: cisco, ios, catalyst
- **VyOS**: vyos

## スキルファイルの追加

新しいベンダーを追加する場合：

1. `skills/ベンダー名/` フォルダを作成
2. マークダウンファイルを作成
3. `main.js` の `detectVendor()` 関数にキーワードを追加

例：
```javascript
if (/juniper|junos/.test(lower)) return 'juniper';
```
